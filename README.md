## About Me

I'm a Full-Stack Developer who enjoys building things that actually get used in the real world — from desktop systems running in production to REST APIs and mobile apps with measurable impact. I work mainly with **JavaScript, TypeScript, React, Angular, Rust and Java**, with a focus on clean architecture, scalability and solutions that solve real problems.

I'm self-taught, adaptable and comfortable jumping into projects at early stages. My current interests include **FastAPI, AI API integration (GPT, MCP) and SaaS product development**. I pick up new technologies fast and prefer learning by building.

- 🖥️ Currently building [StockBeauty](https://github.com/SamuelSanchez17/inventory-management-system) — a desktop POS system in production with CI/CD via GitHub Actions + Tauri autoupdater
- 📱 Built MamLingua, an iOS app actively used by 300+ university students to preserve the Mam indigenous language
- 🔐 Developed a lab access control system processing 500+ monthly records with optimized PostgreSQL queries
- 📫 Reach me at **samuel.sanchezdev17@gmail.com**
